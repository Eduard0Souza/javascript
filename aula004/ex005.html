addEventListener

O addEventListener é um método que permite adicionar eventos a elementos do DOM de forma mais flexível e organizada, utilizando o Javascript em vez do HTML.

Ele permite que você adicione múltiplos ouvintes de eventos a um único elemento, sem sobrescrever os ouvintes existentes.

Exemplo de uso:

      
        var area = document.getElementById('area');
        area.addEventListener('click', clicar);
        area.addEventListener('mouseenter', entrar);
        area.addEventListener('mouseout', sair);
        function clicar(){
            area.innerText = 'Você clicou aqui!';
        }
