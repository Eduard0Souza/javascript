Conversão de tipos

Toda vez que utilizar um prompt, o resultado será uma variável com o tipo de dado String, assim, para fazer uma operação aritimética conforme o exemplo, é necessaário converter a variavel para um tipo de dado number (INT ou Float) utilisando o parse ficando assim: var soma = parseInt(n1) + parseInt(n2); ou seja, pega a variável entre parentes e transforma em um número inteiro. Outra forma de converter o tipo string seria o number utilizado na frente do window, ficando assim: var n1 = Number(window.prompt('Digite um número')) isso ja transformaria o texto digitado em um número.

Da mesma forma existe a Conversão de um número para texto, nesse caso deve-seu utilizar String(n) que transformará o conteúdo entre parenteses em uma variável com o tipo de dado String.

Também pode ser utilizado o n.toString(), tera o mesmo efeito transformando a variável n em uma String

Template String ${var}

Para as últimas versões do JavaScript, foi o criado o template strig, ele substitui a concatenação dos elementos dentro de uma string

Seu use sempre será feito dentro de crases (``), ficando assim: `o Aluno ${var_nome} com ${var_idade} anos tirou nota ${var_nota}`; isso pegará todas as variáveis e colocará nos locais previamente dispostos para elas
