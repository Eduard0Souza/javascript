Comandos JavaScript

Manipulação de dados é uma parte fundamental do JavaScript. Aqui, vamos ver como capturar a entrada do usuário e exibir mensagens personalizadas.

Exemplo de Manipulação de Dados

Use o toFixed() para modificar números, por exemplo, em vez da variável n = 1250.5, pode ser utilizado o n.toFixed(2) que mostrará 2 casas decimais ficando: 1250.50; E tambem pode ser utilizado para fazer substituição de vírgula por ponto, por exemplo, n = 1250.5.toFixed().replace('.', ',') que mostrará 1250,50;

Um comando para utilizar o simbolo da moeda local por exemplo, pode ser dscrito assim: n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}), isso aprensentará o resultado: R$ 1250,50

  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) = real
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'USD'}) = dolar americano
  • n1.toLocaleString('pt-BR', {style: 'currency', currency: 'EUR'}) = EURO
